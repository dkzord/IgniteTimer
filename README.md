
# Ignite Timer

This project serves only as a study of React but is fully functional.


## Install and Running project

Download project
```bash
  git clone https://github.com/dkzord/IgniteTimer.git
```

Installing dependencies

```bash
  pnpm install
```

Run project

```bash
  pnpm run dev
```
